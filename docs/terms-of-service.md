## 3. サービスの説明

GitMemo は、以下の機能を提供するメモアプリケーションです：

- デバイス上でのメモの作成、編集、管理
- Personal Access Token を使用した GitHub Issues とのメモの同期（任意）
- メモのピン留め、整理、ステータス管理（オープン/クローズ）
- アプリ内課金による広告非表示アップグレードの購入

## 4. ユーザーコンテンツ

GitMemo 内で作成されたメモおよびコンテンツ（以下「ユーザーコンテンツ」）の所有権は、お客様に帰属します。本アプリの利用により、お客様は、サービスを提供する目的に限り、お客様のデバイス上でユーザーコンテンツを保存および処理するための限定的なライセンスを当方に付与するものとします。

GitHub 連携を有効にした場合、ユーザーコンテンツ（メモのタイトルおよび本文）は GitHub のサーバーに Issues として送信されます。これは GitHub の利用規約に準拠し、お客様の指示により実行されます。

お客様は、以下に該当するコンテンツを作成しないことに同意するものとします：

- 違法、有害、脅迫的、虐待的、または嫌がらせに該当するもの
- 他者の知的財産権を侵害するもの
- マルウェアまたは有害なコードを含むもの
- 適用される法令に違反するもの

## 5. GitHub 連携

GitHub 連携は任意の機能です。有効にすることにより、お客様は以下に同意するものとします：

- メモの内容が GitHub のサーバーに送信されること
- Personal Access Token のセキュリティについてお客様が責任を負うこと
- GitHub に同期されたメモが GitHub の利用規約およびプライバシーステートメントに準拠すること
- GitHub のサーバーに保存されたデータについて当方が責任を負わないこと

GitHub 連携は、本アプリの設定からいつでも解除できます。

## 7. 広告

GitMemo の無料版は、サードパーティの広告ネットワーク（Google AdMob）が提供するバナー広告を表示します。広告はメモ一覧画面、詳細画面、設定画面に表示されます。メモ入力画面には広告は表示されません。

無料版をご利用いただくことにより、お客様は広告の表示を了承し、同意するものとします。広告非表示アップグレードを購入することで、広告を削除できます。
